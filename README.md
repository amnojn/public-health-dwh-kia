\# Public Health Data Warehouse – Pelayanan Kesehatan \& KIA



\## Overview

This project implements a Medallion Architecture (Bronze–Silver–Gold) Data Warehouse

for Public Health domain, focusing on Healthcare Services and Maternal \& Child Health (KIA).



\## Architecture

\- Bronze Layer: Raw ingestion

\- Silver Layer: Cleansed \& standardized data

\- Gold Layer: Star schema for analytics



\## Tech Stack

\- PostgreSQL

\- SQL

\- Git \& GitHub

\- Draw.io (Data Architecture)



\## Data Source

Synthetic public health datasets (Pelayanan Kesehatan \& KIA)

In progress
